# calicoq version

`calicoq version` shows the version number of the calicoq tool, plus
information about the source code it was built from, and when that was.

## Example

```bash
calicoq version
```

Sample output follows.

```
Version:                   v3.22.0-2.0-calient-0.dev-1530-g7bc5f3aad496
Release:                   v3.22.0-2.0
Open source base version:  v3.31.0
Build date:                2025-11-14T00:01:43+0000
Git commit:                7bc5f3aad496dacd7e9c3a6ab5427a2fc5825f22
```
