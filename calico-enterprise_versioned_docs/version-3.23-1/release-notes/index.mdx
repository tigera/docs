# Calico Enterprise X.Y release notes

:::info early preview release

Calico Enterprise X.Y can be used for previewing and testing purposes only.
It is not supported for use in production.

:::

Learn about the new features, bug fixes, and other updates in this release of $[prodname].

This version of Calico Enterprise is based on [Calico Open Source $[openSourceVersion]](https://docs.tigera.io/calico/$[openSourceVersion]/release-notes).

## New features and enhancements

### Deprecated and removed features

The following features are deprecated and will be removed in an upcoming release:

* Support for the following operating systems:
  * RHEL 8
  * Windows 1809
* Support for the x86-64 v2 chip architecture.
* Application-layer policy and L7 observability features based on Envoy as a daemonset.
  (These are being replaced with similar functionality using Istio Ambient Mode.)
