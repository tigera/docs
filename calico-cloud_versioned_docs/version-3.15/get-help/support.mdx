# Support and feedback

## Contact Support

Before opening a Support ticket, review this checklist of basic [troubleshooting](../get-started/connect/checklist.mdx).

If you still need help, click the chat icon in Manager UI at the bottom of the left nav:

![contact-support](/img/calico-cloud/contact-support.png)

## Provide feedback

We value your feedback and suggestions for improvement. Email us: feedback@calicocloud.io

## Support policy

For details, see our [Support policy](https://www.tigera.io/legal/calico-cloud-support-policy).

## Check the status of Calico Cloud services

You can go to [Calico Cloud Status](https://status.calicocloud.io) to view the current status of our sites and services.
