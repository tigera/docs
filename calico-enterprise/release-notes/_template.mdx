## Upgrading

Before upgrading to Calico Enterprise X.Y, review [Upgrade notes](#upgrade-notes), [Deprecated and removed features](#deprecated-and-removed-features), and [Known issues](#known-issues).

### Upgrade from a previous minor

Calico Enterprise X.Y supports a direct upgrade from version X.(Y−1) and X.(Y−2). To upgrade from an earlier version, plan an intermediate upgrade.

For upgrade instructions, see [Upgrade Calico Enterprise](../getting-started/upgrading/upgrading-enterprise/index.mdx).

### Install a patch release

To install a patch release of Calico Enterprise X.Y, see [Install a patch release](../getting-started/manifest-archive.mdx).
