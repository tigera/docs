# calicoctl review

The `calicoctl review` commands help you audit network policy activity in your cluster.

Read the [calicoctl Overview](../overview.mdx) for a full list of calicoctl commands.

## Subcommands

- [unused-policies](unused-policies.mdx) — List network policies and rules that have not evaluated any traffic within a configurable time window.
