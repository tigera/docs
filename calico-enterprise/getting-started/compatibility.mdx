The following list shows the platforms supported in this release. The tables provide a platform version history, starting with {variables.prodname} 3.5. If you need an earlier version, contact Support.

### Supported {variables.prodname} features

If your platform is listed below, the features in this release will work for your platform unless an individual feature topic explicitly calls out a vendor or {variables.prodname} limitation.

Note that all Windows feature limitations are described in [Windows limitations](install-on-clusters/windows-calico/limitations.mdx), and are not called out in individual Linux topics.
