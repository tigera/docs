
- **AD_train_interval_minutes** - Default: 1440 hours (a full day). 
    Determines the interval between retraining the existing models.

- **AD_search_interval_minutes** - Default: 30 minutes
    Interval between searching for the anomalies.

- **AD_max_docs** - Default: 500000 rows
    Size of the dataset used for the training. The larger the number, the more precise are the trained models, but the more data read from the Elasticsearch storage, and the training takes more time.  

#### Elasticsearch variables

**CLUSTER_NAME** - Default: "cluster". In a multi-cluster deployment, the name of the cluster where the AD job detects the anomalies.
