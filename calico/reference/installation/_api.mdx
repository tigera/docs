<h3 id="operator.tigera.io/v1.ImageSet" class="anchor anchorWithStickyNavbar">ImageSet
<a href="#operator.tigera.io%2fv1.ImageSet" class="hash-link" aria-label="Direct link to ImageSet" title="Direct link to ImageSet">&ZeroWidthSpace;</a></h3>
<div>
<p>
ImageSet is used to specify image digests for the images that the operator deploys.
The name of the ImageSet is expected to be in the format <code>&lt;variant&gt;-&lt;release&gt;</code>.
The <code>variant</code> used is <code>enterprise</code> if the InstallationSpec Variant is
<code>TigeraSecureEnterprise</code> otherwise it is <code>calico</code>.
The <code>release</code> must match the version of the variant that the operator is built to deploy,
this version can be obtained by passing the <code>--version</code> flag to the operator binary.
