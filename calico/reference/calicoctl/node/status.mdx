# calicoctl node status

This sections describes the `calicoctl node status` command.

Read the [calicoctl Overview](../overview.mdx)
for a full list of calicoctl commands.

## Displaying the help text for 'calicoctl node status' command

Run `calicoctl node status --help` to display the following help menu for the
command.

```
View the current status of a Calico node

Usage:
  calicoctl node status [flags]

Flags:
  -h, --help   help for status

Global Flags:
      --allow-version-mismatch   Allow client and cluster versions mismatch
      --context string           The name of the kubeconfig context to use
  -l, --log-level string         Set the log level (panic, fatal, error, warn, info, debug) (default "panic")
```

### Examples

Check the status of a $[prodname] instance.

```bash
sudo calicoctl node status
```
