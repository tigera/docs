---
title: kubectl commands for policy
description: Commands to view policies in default and non-default tiers.
---

Use `kubectl` to view network policies in your managed Kubernetes clusters in the default tier.

**List network policies**

```bash-plain-text
kubectl get -n <namespace> networkpolicy.p
```

**List global network policies**

```bash-plain-text
kubectl get globalnetworkpolicy
```

**List staged network policies**

```bash-plain-text
kubectl get -n <namespace> stagednetworkpolicy
```

**List global staged network policies**

```bash-plain-text
kubectl get stagedglobalnetworkpolicy
```

To view any policies in a non-default tier, include `-l projectcalico.org/tier=<tier-name>` in the command.
